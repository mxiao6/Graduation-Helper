## Instruction

* Install
  ```bash
  npm install && cd client && npm install
  ```

* Run
  ```bash
  npm run dev
  ```
  
* Fix style
  ```bash
  semistandard --fix "routes/**/*.js" "*.js"
  semistandard --fix --env mocha "test/**/*.js"
  ```
